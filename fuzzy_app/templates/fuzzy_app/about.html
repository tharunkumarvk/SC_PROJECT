About This Project

Disease Prediction using Fuzzy Logic

This system uses fuzzy logic to predict possible diseases based on symptom inputs. Fuzzy logic allows for handling imprecise or uncertain information, which is common in medical diagnosis.

How It Works
  1. Enter your symptom values in the input form
  2. The system calculates membership degrees for each symptom in fuzzy sets (low, medium, high)
  3. Fuzzy rules are applied to determine possible diseases
  4. Results are displayed with confidence levels
Technologies Used
  • Django - Python web framework
  • Fuzzy Logic - For handling uncertain medical information
  • Bootstrap - For responsive user interface
